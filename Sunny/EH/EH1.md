GUI (Graphic user interface)

enhance의 분석 스크린의 경우 3가지 층으로 나뉨 !
left panel : 데이터에 존재하는 변수목록 보여줌!
middle panel : 왼쪽 패널에서 선택한 변수의 요약 통계를 보여주며, 원하는 변수에 따라 다른 통계를 보여줌!
right panel: 두 변수의 결과에 따라 그룹을 선택하고 수정할 수 있음, 그룹화하려는 변수 선택 가능, 작업별로 그룹에서 선택한 변수에 대한 선택사항을 보여주며, 핵심증진기의 강화된 작업을 사용하는 데 필요한 데이터로부터 통찰력을 얻기 위해 머신러닝 알고리즘을 사용하여 관계를 보여주며 데이터로부터 통찰력을 얻음!

강화버튼 클릭하면 모델에 포함될 변수 중에서 선택할 수 있는 변수 목록을 얻을 수도 있고 ok를 누르고 변수를 선택하면 몇 초안에 데이터를 통해 코드 학습을 할 수 있고  쉽게 읽을 수 있는 다채로운 사용자 인터페이스 결과 패널에서 세그먼트 미리보기, 예측이 포함된 세가지 주요 휠이 포함됨!

예를 들어 어떤 데이터에서 기계 학습 알고리즘에 의해 생성된 세그먼트의 공통 특징을 보기 위해 각 세그먼트를 클릭할 수 있고 따라서 세그먼트의 손익률을 확인할 수 있음 (평균과 비교한 이득과 손해)

그 후 중간 섹션에서 클릭하면 세그먼트의 레코드 목록을 볼 수 있고 엑셀 파일로 내보낼 수 있음. 만약 3을 클릭하면 실제 트리와 매우 유사한 인터페이스를 얻고 끝까지 도달할 대까지 더하기 기호를 클릭하면 데이터가 어떻게 분할되는 지도 볼 수있음(트리맵 구조)
```

```
데이터 임포트 

프로젝트와 워크스페이스를 유저가 만들면 인터페이스가 나오는데 해당 프로젝트에 +를 누르면 데이터 임포트 창이 뜨게됨 .. 그런 후에 엑셀파일,csv, txt, typeform, 구글폼스, 오라클 db, mysql(mariadb), sql server 등의 형식을 지원하며 데이터 임포트 및 설정을 한 후에 분석 인터페이스를 보여줌!

예측 

분석 인터페이스에서 가장 오른 쪽 항목에서 선택할 수 있는 질문과 범위가 있음 
ex) 어느 나라인가, 연랭대, 마진, 그외 질문이 있음
완료가 되었다면 미들 패널에 예측 버튼을 클릭하면 분석치를 확인할 수 있음 
분석치의 경우 export가 가능함!

예측 관련 소개

(ex: 체리 아이스크림은 어떤 사람에 효율성이 좋은지?)
